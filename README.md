# ChatGPT

This repository contains **ChronoQuest: Team Showdown** — a free-play slot
machine demo (Phaser + TypeScript + Vite).

The project lives in [`chronoquest-slot/`](./chronoquest-slot).

```bash
cd chronoquest-slot
npm install
npm run dev
```

See [`chronoquest-slot/README.md`](./chronoquest-slot/README.md) for full docs.

> Free-play demo only — fake credits, not real-money gambling.
